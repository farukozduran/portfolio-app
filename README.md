# 🚀 Personal Portfolio Website (React + .NET)

A modern full-stack personal portfolio website built with **React** and **.NET Web API**, designed to showcase my projects, skills, and backend development experience.

---

## 📌 Overview

This project is a fully responsive portfolio website where I present my work as a **Backend Developer (.NET)**.  
It includes real backend integration, clean UI design, and a scalable structure.

---

## 🎯 Why This Project?

This project was built to demonstrate:

- Real-world backend + frontend integration
- API-driven architecture using .NET
- Clean and scalable UI design with React
- Ability to build and structure a full-stack application


## 🛠️ Tech Stack

### 🔹 Frontend
- React
- React Router DOM
- Custom CSS (no Tailwind)
- CSS Variables (Design System)

### 🔹 Backend
- .NET Web API
- RESTful API
- C#

### 🔹 Other
- Git & GitHub
- Responsive Design
- Dark Mode Support

## ✨ Features

- 📄 Multi-page routing (Home, About, Projects, Contact)

## ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/farukozduran/portfolio-app.git
```

### 2️⃣ Backend Setup (.NET)
```bash
cd backend
dotnet restore
dotnet run
```

API runs on:

```
http://localhost:5031/api/projects
```
### 3️⃣ Frontend Setup (React)

```bash
cd frontend
npm install
npm run dev
```
---

## ⚙️ Environment Variables

Create a `.env` file in the frontend folder:

```env
VITE_API_URL=http://localhost:5031
```
